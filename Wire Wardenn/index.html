WIRE WARDEN

Networking the Fun Way.

A 2D pixel game about configuring and protecting network cables

OBJECTIVES

Build real-world networking skills through interactive gameplay.

Players learn LAN cabling and basic network setup through hands-on challenges.

IDENTIFY

Identify and understand LAN cable types.

APPLY

Apply correct RJ45 color coding (T568A & T568B).

SIMULATE

Simulate proper LAN cable assembly.

UNDERSTAND

Understand basic router and switch setup.

Game Features

Pixel-art visual style with a pastel color palette

Interactive cable wiring simulation

Router and switch configuration challenges
